Users CRUD API

Welcome to the Users CRUD Application built with Jakarta EE 10 and Clean Architecture.

API Endpoints

GET /api/users
Get all users. Use ?active=true to filter active users only.
GET /api/users/{id}
Get a specific user by ID.
GET /api/users/email/{email}
Get a user by email address.
POST /api/users
Create a new user. Requires JSON body with name, email, phone (optional), and active fields.
PUT /api/users/{id}
Update an existing user. Requires JSON body with name, email, phone, and active fields.
DELETE /api/users/{id}
Delete a user by ID.
